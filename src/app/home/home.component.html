Patient List Page

This is patient listing page, Click on the button to fetch the patient list. Few things have to be doen in order to make things possible are as follows:

  • The https://try.smilecdr.com/baseR4/Patient giving CORS issue, so implemented proxy in order to complete the task.
  • Implemented the loading on every http call, use interceptor to make it look good.
  • Upon fetching the data you can see the results in the table format and the table is sorted on First name and Birthdate.
  • The fetched results displays on the Patient-List component rather then in template.
